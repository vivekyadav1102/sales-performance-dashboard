# Sales Performance Dashboard

## Project Overview

This project focuses on analyzing sales data to uncover key business insights and improve decision-making.
An interactive dashboard was built using Power BI, supported by data cleaning and validation in SQL.

## Objectives

* Analyze overall sales and profit performance
* Identify top-performing regions, categories, and products
* Detect loss-making areas
* Provide actionable business insights

## Tools & Technologies

* SQL (Data Cleaning & Validation)
* Power BI (Data Visualization & Dashboard)
* Excel/CSV (Data Source)

## Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Total Orders

## Dashboard Features

* Sales trend analysis over time
* Region-wise performance comparison
* Category and sub-category profitability
* Top-performing products identification
* Interactive filters (Region, Category, Date)
* Scatter plot for Sales vs Profit analysis

## Key Insights

* Sales show seasonal fluctuations indicating demand patterns
* West region is the top-performing region in terms of sales
* Profit margin is approximately 12%, indicating stable performance
* Technology category generates the highest profit
* Some sub-categories incur losses despite good sales
* A small number of products contribute significantly to total revenue
* Higher sales do not always result in higher profit

## Learning Outcomes

* Hands-on experience with SQL data cleaning and validation
* KPI creation and business analysis
* Building interactive dashboards using Power BI
* Understanding the difference between aggregated vs row-level calculations
